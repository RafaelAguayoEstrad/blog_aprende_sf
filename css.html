CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para definir el estilo y la presentación de documentos HTML. Permite controlar aspectos como el color, el diseño, la tipografía y la disposición de los elementos en una página web.

Conceptos importantes de CSS incluyen:

  • Selectores CSS: permiten aplicar estilos a elementos específicos en una página.
  • Propiedades CSS: controlan aspectos como color, tamaño, margen, relleno, etc.
  • Modelo de caja: describe cómo se renderizan y se colocan los elementos en una página.
  • Posicionamiento: controla la ubicación y el flujo de los elementos en la página.
  • Flexbox y Grid Layout: proporcionan herramientas poderosas para crear diseños complejos y responsivos.

Aquí tienes un ejemplo básico de cómo utilizar CSS para cambiar el color de fondo de un elemento:

/* CSS externo */
body {
    background-color: #f0f0f0;
}

/* CSS interno */
<style>
    p {
        color: blue;
    }
</style>

/* CSS en línea */
<p style="font-size: 18px;">Hola, mundo!</p>
