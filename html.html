HTML

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear la estructura y el contenido de las páginas web. Define los elementos y su estructura básica.

Conceptos clave de HTML incluyen:

  • Etiquetas HTML: elementos como <p>, <div>, <h1>, <img>, etc., que definen el contenido y la estructura de la página.
  • Atributos: proporcionan información adicional sobre los elementos, como href en los enlaces o src en las imágenes.
  • Formularios: elementos como <form>, <input>, <select>, etc., para recopilar datos del usuario.
  • Tablas: elementos para mostrar datos en filas y columnas.
  • Elementos semánticos: <header>, <footer>, <article>, <section>, etc., para definir la estructura de la página de manera más significativa.

Aquí tienes un ejemplo básico de código HTML:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Página Web</title>
</head>
<body>
    <h1>Hola, mundo!</h1>
    <p>Esta es mi primera página web.</p>
</body>
</html>

Además de los elementos mencionados anteriormente, HTML también ofrece etiquetas para multimedia, enlaces, metadatos, accesibilidad y mucho más.
